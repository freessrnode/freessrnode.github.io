什么是 Clash？

Clash 是一款开源的基于规则的代理工具，广泛应用于科学上网领域。它支持多种代理协议，包括 Shadowsocks、Vmess、Trojan 等，以及更多的自定义协议，帮助用户突破网络封锁，保护上网隐私。通过灵活的规则配置，用户可以根据不同需求，选择不同的代理服务器或线路，提高网络速度，优化上网体验。

Clash 提供了非常强大的功能，用户不仅可以选择不同的代理协议，还可以通过配置文件或订阅链接的方式，动态管理和更新代理服务器的列表。它支持跨平台使用，覆盖 Windows、Mac、Linux、Android 和 iOS 等多种操作系统，确保用户在各种设备上都能获得稳定、快速的网络连接。

什么是 Clash 订阅链接？

Clash 订阅链接是一种特殊的 URL，通过这个链接，用户可以自动导入并更新其代理配置。当用户订阅某个代理服务时，服务商会提供一个订阅链接，用户可以将这个链接导入到 Clash 客户端中。Clash 会定期从该链接获取最新的代理配置，无需手动干预，就能够自动更新节点信息或规则。

通过订阅链接，用户不仅能够轻松切换代理服务器，还能随时获取服务商的最新配置，确保网络连接始终保持最佳状态。无论是切换服务器节点，还是调整代理规则，订阅链接都能让整个过程变得更加高效和便捷。

Clash 支持 Windows、Mac、Linux、Android 和 iOS 等多个平台，用户可以在各种设备上使用它。

FAQ

Shadowrocket 的重写功能有什么用途？
Shadowrocket 的重写功能允许用户在请求阶段修改 HTTP 头、响应内容或重定向 URL。这一功能常用于屏蔽广告、解锁区域限制或调试网页请求，非常灵活强大。
Nekoray 节点订阅自动更新如何配置？
用户可添加订阅 URL，Nekoray 会定期下载并替换旧节点。这样可保证节点列表始终最新，减少手动维护，提高使用效率，同时保证网络访问稳定可靠。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
Shadowrocket 能否自动更新订阅？
支持。Shadowrocket 可以设置定时更新周期，例如每隔 12 小时或每天自动刷新订阅链接，确保节点列表保持最新状态。这样无需手动更新即可长期稳定使用。
Nekoray 节点订阅自动更新如何设置？
用户可添加订阅 URL，Nekoray 会定期自动下载并更新节点列表。保持节点最新可减少手动维护，提高节点可用性和网络连接稳定性，确保科学上网体验顺畅。
Mellow 能与 Clash 共存吗？
可以。Mellow 与 Clash 核心不同，前者采用透明代理技术，后者通过系统代理生效。两者配置端口不冲突的情况下可共存运行，用于不同的流量处理场景。
V2Ray 的 WebSocket + TLS 配置有哪些优势？
WebSocket + TLS 可以伪装成普通 HTTPS 流量，增强抗封锁能力。适合高封锁环境，结合域名伪装和策略组，可实现高安全性和稳定的网络访问体验。
ShadowsocksR 支持哪些混淆插件？
ShadowsocksR 提供多种混淆方式，如 tls1.2_ticket_auth、http_simple、random_head 等。它们通过模拟正常流量特征来规避防火墙检测，使通信更加隐蔽。
Surfboard 的配置文件可以与 Clash 通用吗？
Surfboard 基于 Clash 核心，因此支持大部分 YAML 配置文件，但部分字段如 rule-provider、external-controller 等可能不兼容。建议使用订阅转换工具将 Clash 文件转换为 Surfboard 兼容格式导入。
Clash 如何手动设置某应用直连？
在 Clash 的规则文件中添加 PROCESS-NAME 规则，如 `PROCESS-NAME,chrome.exe,DIRECT`。保存并重启后，该应用的流量将绕过代理直连。适用于游戏或内网程序等特殊场景。
